My `neovim` configuration
---------------------

Installation steps
===================

1) Clone to your home folder:

```
	git clone git@github.com:ildus/nvim.git ~/.config/nvim
```

2) Start neovim
3) Run `:PackerSync`
4) Install `ripgrep`, `fd`, `python-neovim`
5) Enter <space> or <leader> or <g> to see mappings in NORMAL mode.
